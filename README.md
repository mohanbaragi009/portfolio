## 🚀 Getting Started (Local Setup)

### Prerequisites
Make sure you have the following installed:
- **Node.js** (v14 or above) → [Download here](https://nodejs.org/)
- **Git** → [Download here](https://git-scm.com/)

### Step 1 — Clone the Repository
```bash
git clone https://github.com/mohanbaragi009/portfolio.git
cd portfolio
```

### Step 2 — Install Dependencies
```bash
npm install
```

### Step 3 — Start the Development Server
```bash
npm start
```

> Opens at [http://localhost:3000](http://localhost:3000) in your browser.  
> The page auto-reloads when you save changes.

### Step 4 — Build for Production
```bash
npm run build
```
> Creates an optimized production build in the `/build` folder.

## ☁️ Deployment on Vercel

This portfolio is deployed using **[Vercel](https://vercel.com)** — zero config, production-grade hosting.

### Step 1 — Push code to GitHub
```bash
git add -A
git commit -m "your message"
git push origin master
```

### Step 2 — Import to Vercel
1. Go to [vercel.com/new](https://vercel.com/new)
2. Click **"Add New Project"**
3. Select the **`mohanbaragi009/portfolio`** repository
4. Vercel auto-detects it as a **Create React App** ✅

### Step 3 — Deploy Settings (leave as default)
| Setting | Value |
|---|---|
| Framework | Create React App |
| Build Command | `npm run build` |
| Output Directory | `build` |
| Install Command | `npm install` |

### Step 4 — Click Deploy 🎉
You'll get a live URL in ~2 minutes!  
Every `git push` automatically triggers a **re-deploy**.

## 📝 Customization Guide

To update the portfolio with your own information:

| File | What to change |
|---|---|
| `src/components/Home/Home.js` | Your name, social links |
| `src/components/Home/Home2.js` | Intro paragraph text |
| `src/components/Home/Type.js` | Typewriter roles/titles |
| `src/components/About/AboutCard.js` | Bio, location, college, activities |
| `src/components/About/Techstack.js` | Your tech skills |
| `src/components/About/Toolstack.js` | Your tools |
| `src/components/Projects/` | Your project cards |
| `src/components/Resume/` | Your resume details |
| `src/Assets/` | Replace images and resume PDF |
